## Overview

ISO/IEC 42001:2023 is the international standard for AI Management Systems (AIMS). It defines requirements for establishing, implementing, maintaining, and continually improving an AIMS within an organization.

**Applies to:** `actor: provider` (any risk level)

**19 assertions** across Clauses 4–10.

## Relationship to EU AI Act

ISO 42001 is complementary, not redundant. The EU AI Act specifies *what* documentation must exist; ISO 42001 specifies the *management system* that produces and maintains it. Running both frameworks in Rulestatus checks both the artifacts and the system behind them.
