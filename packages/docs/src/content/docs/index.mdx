## Install

```bash
bun install -g rulestatus
```

## Five-minute start

```bash
rulestatus init           # creates .rulestatus.yaml
rulestatus generate --all # scaffolds all required docs
# fill in the TODOs
rulestatus run            # see what's missing
```

> **Disclaimer:** Rulestatus checks whether required documentation and configuration is present and correctly structured. Evidence present does not constitute a legal determination of compliance. Conformity assessment for high-risk AI systems may require evaluation by a notified body. Treat reports as due diligence documentation, not compliance certificates.
