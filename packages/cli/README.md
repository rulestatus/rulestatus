# @rulestatus/cli

**ESLint for AI law.** Executable compliance testing for EU AI Act, ISO 42001, NIST AI RMF, and Colorado SB 24-205 — runs in CI, produces audit-grade attestations.

```bash
npm install -g @rulestatus/cli
rulestatus init && rulestatus run
```
